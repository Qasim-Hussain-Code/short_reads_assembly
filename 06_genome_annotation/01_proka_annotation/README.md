# Prokka Annotation Results

This directory contains genome annotation results from Prokka v1.15.6.

## Output Files
- `codanics_prokka.gff` - Annotation in GFF3 format
- `codanics_prokka.gbk` - Annotation in GenBank format
- `codanics_prokka.faa` - Protein sequences (FASTA)
- `codanics_prokka.ffn` - Nucleotide sequences of features (FASTA)
- `codanics_prokka.fna` - Input genome sequence
- `codanics_prokka.tsv` - Tab-separated annotation table
- `codanics_prokka.txt` - Summary statistics
- `codanics_prokka.err` - Error/warning log
- `codanics_prokka.log` - Run log
- `codanics_prokka.sqn` - Sequin format for NCBI submission
- `codanics_prokka.tbl` - Feature table

## Summary
- CDS: ~4,500
- tRNA: 80
