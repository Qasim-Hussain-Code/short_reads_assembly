# Genome Quality Assessment

This directory contains quality assessment results for the assembled genome.

## Subdirectories
- `01_checkm2/` - CheckM2 completeness and contamination results
- `02_quast/` - QUAST assembly statistics
- `03_quast_busco_others/` - QUAST extended analysis with gene finding
- `04_busco_assessment/` - BUSCO gene completeness results

## Key Results
- CheckM2: Completeness and contamination metrics
- QUAST: N50, total length, number of contigs
- BUSCO: 98.3% complete (bacteria_odb12)
